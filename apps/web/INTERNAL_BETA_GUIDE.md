# AgentAPI 内测使用手册

## 开始聊天

1. 打开 AgentAPI Web 页面并登录。
2. 点击左侧 `新聊天`。
3. 在底部输入框输入问题，点击发送按钮。
4. 新对话只有在发送第一条消息后才会出现在左侧会话列表中。

## 上传文件

1. 点击输入框左侧的上传按钮。
2. 选择本地文件，等待文件名显示在输入框上方。
3. 输入你的问题并发送。
4. 上传后的文件可在左侧 `Attachments` 中查看、预览或删除。

## 生成图片

1. 仅在新对话空白状态下点击 `生成图片`。
2. 输入图片需求并发送。
3. 生成任务会作为 Job 执行，完成后结果会出现在 `Artifacts`。
4. 点击 `Artifacts` 会打开 Artifact Workspace，可预览、下载或删除结果。

## 查看 Artifact

1. 点击左侧 `Artifacts`。
2. 在 Artifact Workspace 中选择一个文件。
3. 可使用顶部按钮打开预览、下载、提取 Memory 或删除。
4. 新 artifact 产生后，`Artifacts` 外侧会出现提示；点击查看后提示消失。

## 使用 Skills

1. 点击左侧 `Skills` 查看可用技能。
2. 选择技能后会把对应命令插入输入框。
3. 补全参数后发送。
4. 长任务会进入 `Jobs`，产物会进入 `Artifacts`。

## 管理 Memory

1. 点击左下角设置按钮。
2. 选择 `Manage Memory`。
3. 可查看、编辑、删除 Memory，也可调整 Memory 开关。
4. 删除 Memory 是不可恢复操作，内测时请先确认目标条目。

## Live 注意事项

1. Live 需要浏览器麦克风权限和服务端 Vertex Live 凭证。
2. 切换回 Chat 会停止麦克风采集。
3. 可分别调整扬声器音量和麦克风音量；麦克风音量为 0 时会暂停采集。
4. 如果看到“Live mode is not configured”，说明当前环境未完成 Live 凭证配置，请联系管理员。
5. 网络断开或权限被拒绝时，页面会给出用户可读提示，不需要刷新整个浏览器即可回到 Chat。

## 内测反馈

反馈问题时请提供：浏览器、页面地址、会话标题、发生时间、操作步骤和截图。不要在截图或文字中粘贴密钥、token、服务账号 JSON 或内部日志全文。
